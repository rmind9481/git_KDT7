## 평면에 적을 그려주는 프로그램 설계

- 사용자가 원하는 위치에 점을 표시

- 점을 표현/저장하는 데이터 타입 필요

- 새로운 데이터 타입 설계
	* 속성/특징/외형 : x좌표,y좌표, 색상, 반지름 ==> 코드화 ==> 변수명 x,y, color, r
	* 행동/기능/역할 : 점그리기, 정보출력 		==> 코드화 ==> 명수명 drawing(), printinfo()
	* 데이터타입 이름 : point
	* 부모/슈퍼클래스 

	* 속성/특징/외형 : x좌표,y좌표,z 좌표 색상, 반지름 ==> 코드화 ==> 변수명 x,y,z color, r
	* 행동/기능/역할 : 점그리기, 정보출력 		==> 코드화 ==> 명수명 drawing(), printinfo()
	* 데이터타입 이름 : point3D
	* 자식/서브클래스
